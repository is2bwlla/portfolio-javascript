Entendendo a window e o document.

WINDOW:

A window em JavaScript é um objeto global que representa a janela do navegador. Ele fornece acesso a várias funcionalidades essenciais, como:

Manipulação do DOM:

Permite acessar e modificar elementos da página.

Eventos:

Facilita o registro e a gestão de eventos, como cliques e movimentos do mouse.

Armazenamento:

Oferece APIs como localStorage e sessionStorage para armazenamento de dados no navegador.

Navegação:

Permite controlar o histórico do navegador, redirecionar a URL e abrir novas janelas ou abas.

Timers:

Fornece métodos como setTimeout e setInterval para agendar execução de funções.

Em resumo, a window é fundamental para a interação com a interface do usuário e para a manipulação de comportamentos no ambiente do navegador.

EXEMPLOS:

1. window.alert:

O alert serve para exibir uma mensagem de alerta na tela.

2. window.confirm:

O confirm serve para exibir uma mensagem de confirmação ou cancelamento.

3. window.prompt:

O prompt serve para exibir um input na tela.

4. document.write:

O write serve para adicionar texto diretamente no meu HTML, na minha tela de exibição.

5. Number:

O Number é usado para especificar que input recebeerá um valor numérico, dessa forma ele consegue fazer contas e dar resultados corretos ao invés de fazer uma concatenação dos valores, como seria caso ele identificasse o input como uma String.
