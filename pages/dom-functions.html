DOM e Functions:

O que é o DOM?


O DOM (Document Object Model) é uma interface de programação que representa a estrutura de um documento HTML ou XML como uma árvore de nós, onde cada elemento, atributo e texto no documento é um nó que pode ser acessado e manipulado via JavaScript. Isso permite que os desenvolvedores modifiquem dinamicamente o conteúdo, a estrutura e o estilo de uma página web em resposta a interações do usuário, sem precisar recarregar a página inteira.

O que são o Functions?

São blocos de códigos reutilizáveis que executam uma tarefa específica ou calcula um valor. Elas servem para organizar o código, permitindo que você agrupe operações relacionadas e estruture seu cósigo a fim de evitar repetições.

FUNÇÃO DECLARADA: ideal para funções globais.
EXPRESSÃO DE FUNÇÃO: útil para funções usadas localmente.
FUNÇÃO DE SETA: melhor para callbacks curtos, funções anônimas e simplificação de sintaxe.
