POO e Arrays:

O que é POO?


POO (Programação Orientada a Objetos) em JavaScript é um paradigma de programação que organiza o código em torno de objetos, que são instâncias de classes. Objetos contêm propriedades (dados) e métodos (funções) que representam o estado e o comportamento do objeto, respectivamente.

Os principais conceitos da POO em JavaScript são:

1. Classes: Moldes para criar objetos com atributos e métodos.
2. Objetos: Instâncias de uma classe que possuem propriedades e métodos.
3. Herança: Permite que uma classe derive de outra, reutilizando e estendendo seu comportamento.
4. Encapsulamento: Protege o estado interno do objeto, permitindo acesso controlado.
5. Polimorfismo: Permite que diferentes classes tenham métodos com o mesmo nome, mas comportamentos diferentes.
6. Abstração: Simplificação de conceitos complexos para facilitar o uso de objetos.

Em JavaScript, classes podem ser definidas usando a palavra-chave class, introduzida no ES6.
